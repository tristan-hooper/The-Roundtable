# The-Roundtable
The Roundtable is a self-refinement evolution prompt built for LLM's that creates four distinct AI entities: (🔒Summa), (🔓Imaginatio), (📝Arkhitekton), and (🧠Cogitator). Each of these has a different role that they play, and utilize psychological and sociological learning strategies for feedback loop of continual growth and development.
